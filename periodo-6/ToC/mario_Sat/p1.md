proposicional 𝜙, escrita em FNC, se tal atribuição existir. Você só

como uma função do tamanho da fórmula. Dica: Descreva como
fixar uma variável resulta em uma fórmula menor nas variáveis
restantes.

R: Atribua um valor para uma variavel arbitraria que ainad não tenhaum valor, e pergunte ao oráculo se a nova formula é satisfativel, se a respost for sim mantenha o valor atribuio e execute novamente o algoritimo, se a respota for não inverta o valor atribuido e execute o algoritmo novamente, quando não houver mais variáveis para serem atribuidas pare o algoritimo e retorne as atribuições das variaveis.

Esse algoritmo faz uma quantide de perguntasao oráculo linearmente em função do tamanho da formula, fazendo sempre n perguntas onde é a quantiadde 

``` code
alg(𝜙)
    if existe uma variável x sem atribuição de valor
        x <- Verdadeiro
    else
        return atribuição das variáveis 
    
    resposta <- pergunte ao oráculo
    
    if resposta == Verdadeiro
        alg(𝜙)
    if resposta == Falso
        x <- Falso        
        alg(𝜙)
```

### Exercicio 5

> Suponha que o mesmo oráculo possa resolver o problema de decisão
𝑘−Coloração de Grafos. Mostre que perguntando um número
polinomial de questões, você pode encontrar uma 𝑘−coloração
se ela existir. Dica: Como nós podemos modificar o grafo para
indicar que dois vértices são da mesma cor, ou que eles são de cores
diferentes?

R:
Unimos dois vertices quaisquer que não são vizinhos, perguntamos ao oráculo, se a resposta for sim, esses dois são damesma cor e podemos proseguir chamando o algoritmo no novo grafo, se a resposta for não, escolhemos outros dois vertices para executar a união. Quando não existir mais dois vertices que podem ser unidos o algoritmo paramos retornand os conjuntos formados. Cada conjunto representa uma cor diferente que pode ser usada para colorir o grafo.
